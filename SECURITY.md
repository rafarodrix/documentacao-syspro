## Reportar vulnerabilidade

**Não abra uma issue pública.**

- GitHub Security Advisories: aba "Security" → "Report a vulnerability"
- E-mail direto ao time de plataforma

Áreas sensíveis: autenticação (`better-auth`, `agentToken`), controle de acesso por tenant, acesso remoto (RustDesk), dados fiscais (NF-e, CNPJ).
