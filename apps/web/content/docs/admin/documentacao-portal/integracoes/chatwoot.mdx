# Integração: Chatwoot

> Customer support platform integrada ao portal Trilink.
> Atualizado em: 2026-05-05

---

## O que é

Chatwoot é a plataforma de atendimento ao cliente usada para gerenciar conversas com clientes. A integração é **bidirecional**: eventos do Chatwoot são recebidos via webhook e o portal pode criar/atualizar conversas no Chatwoot.

---

| Arquivo (apps/api)                                           | Responsabilidade                        |
|--------------------------------------------------------------|-----------------------------------------|

## Configuração

```bash
CHATWOOT_API_URL=https://chatwoot.example.com
CHATWOOT_API_KEY=api_token_aqui
```

Configuração por empresa disponível em `/portal/configuracoes?tab=chatwoot`.

---

## Webhook: Chatwoot → Portal

**Endpoint:** `POST /api/integrations/chatwoot/webhook`

Eventos processados:

| Evento Chatwoot            | Ação no portal                                        |
|----------------------------|-------------------------------------------------------|
| `conversation_created`     | Cria/atualiza `Conversation` no banco                 |
| `conversation_updated`     | Sincroniza status e assignee                          |
| `message_created`          | Persiste `ConversationMessage`                        |
| `conversation_resolved`    | Atualiza status da conversa                           |
| `csat_survey_completed`    | Persiste `ChatwootCsatRating`                         |

---

## Portal → Chatwoot

O `chatwoot.client.ts` expõe métodos HTTP:

```typescript
// Criar conversa no Chatwoot
await chatwootClient.createConversation({
  accountId,
  inboxId,
  contactId,
  message: '...',
})

// Atualizar status
await chatwootClient.updateConversation(conversationId, {
  status: 'resolved',
})

// Adicionar label
await chatwootClient.addLabel(conversationId, 'suporte-tecnico')
```

## CSAT (Customer Satisfaction)

Fluxo de CSAT implementado:

1. Conversa é encerrada no Chatwoot
2. Chatwoot envia pesquisa CSAT ao cliente
3. Webhook `csat_survey_completed` chega ao portal
4. Portal persiste rating em `ChatwootCsatRating`
5. Dashboard exibe métricas de satisfação

**Force resolve:** O portal pode forçar a resolução de conversas para disparar CSAT.

---

## SSO com Chatwoot

Agentes de suporte podem fazer SSO no Chatwoot via portal:

**Endpoint:** `GET /api/users/me/chatwoot/sso`

Gera token de SSO do Chatwoot e redireciona o agente autenticado diretamente para a conversa.

## Relacionamento com Evolution (WhatsApp)

A integração Chatwoot e Evolution é sincronizada:

```
WhatsApp (Evolution) ──► portal ──► Chatwoot
                                        │
                                    agente responde
                                        │
Chatwoot ──► webhook ──► portal ──► Evolution ──► WhatsApp
```

O portal é o hub central de sincronização — nunca há comunicação direta Evolution ↔ Chatwoot.
