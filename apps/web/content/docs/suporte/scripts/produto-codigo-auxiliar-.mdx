## Objetivo

Este script realiza a atualização em massa do campo `cd_auxiliar` na tabela `produto`. Ele é útil para padronizar a codificação auxiliar quando há furos na sequência ou quando se deseja iniciar uma nova numeração a partir de um valor específico (neste caso, 1000).

> **Atenção:** Este script altera **todos** os registros da tabela de produtos. O backup da base de dados é obrigatório antes da execução.

## Script SQL (Firebird 5.0)

```sql
-- Script: Atualizar a sequência do código auxiliar dos produtos
-- Objetivo: Reiniciar a numeração a partir de 1000 seguindo a ordem do ID.

MERGE INTO produto p_target
USING (
  SELECT
    p.id_produto,
    999 + ROW_NUMBER() OVER (ORDER BY p.id_produto) AS novo_cd_auxiliar
  FROM produto p
) p_source ON (p_target.id_produto = p_source.id_produto)
WHEN MATCHED THEN
  UPDATE SET p_target.cd_auxiliar = p_source.novo_cd_auxiliar;

COMMIT;
```
