## Introdução

Para garantir a conformidade com a Base Índice Nacional (BIN) e a correta emissão de documentos fiscais, o Syspro ERP utiliza tabelas padronizadas em seus campos de seleção. Este guia detalha as opções disponíveis em cada menu suspenso, auxiliando o usuário na classificação precisa do ativo.

<Callout type="info">
**Dica Operacional:** A classificação correta da Categoria e do Tipo de Carroceria é cruzada pela SEFAZ com os dados do RNTRC do transportador.
</Callout>

## Tabela de Tipos de Veículo (BIN/RENAVAM)

Conforme a padronização DENATRAN, utilize os códigos abaixo para o campo **Tipo Veículo**:

| Grupo | Códigos e Descrições | Registro BIN |
| :--- | :--- | :--- |
| **Carga/Tração** | 14-Caminhão, 17-Caminhão Trator, 23-Caminhonete | Sim |
| **Passageiros** | 06-Automóvel, 07-Microônibus, 08-Ônibus | Sim |
| **Reboques** | 10-Reboque, 11-Semi-Reboque | Sim |
| **Não Motorizados** | 01-Bicicleta, 12-Charrete, 15-Carroça | Não |

## Perguntas Frequentes

<Accordions>
<Accordion title="O que significa 'Não cadastrado na BIN' no Tipo de Veículo?">
Significa que o veículo (como bicicletas ou carroças) não possui registro na Base Índice Nacional de veículos automotores, sendo usado apenas para controle interno ou logístico simples.
</Accordion>

<Accordion title="Quando devo marcar o VIN como 'Remarcado'?">
Apenas quando o veículo tiver passado por processo de regravação de chassi autorizado pelo DETRAN, geralmente após roubo/furto com dano ao chassi original ou corrosão severa.
</Accordion>

<Accordion title="O campo 'Tipo de Restrição' bloqueia o uso do veículo no sistema?">
Não. Ele tem caráter informativo para que o gestor de frota saiba que o veículo está vinculado a um contrato de financiamento ou penhor.
</Accordion>
</Accordions>
