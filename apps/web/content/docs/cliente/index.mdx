## Navegue por objetivo

Use esta central conforme o tipo de necessidade: aprender o fluxo, executar uma rotina, resolver uma duvida pontual ou acionar apoio operacional.

<Cards>
  <Card title="Primeiros Passos" href="/portal/docs/cliente/primeiros-passos">
    Fluxos iniciais para ambientacao, acesso e navegacao pelas areas principais do sistema.
  </Card>
  <Card title="Documentacao" href="/portal/docs/cliente/documentacao">
    Guias operacionais organizados por modulo, com rotinas completas e parametrizacoes do dia a dia.
  </Card>
  <Card title="Duvidas" href="/portal/docs/cliente/duvidas">
    Respostas objetivas para erros comuns, rejeicoes da SEFAZ, integracoes e duvidas recorrentes.
  </Card>
  <Card title="Treinamentos" href="/portal/docs/cliente/treinamentos">
    Trilhas praticas por area para capacitacao e apoio a novos usuarios.
  </Card>
  <Card title="Suporte ao cliente" href="/portal/docs/cliente/suporte">
    Orientacoes sobre atendimento, requisitos e canais para tratar incidentes e solicitacoes.
  </Card>
</Cards>

<Callout title="Como usar esta central">
  Comece por <strong>Primeiros Passos</strong> se voce estiver iniciando. Use <strong>Documentacao</strong> para executar processos completos, <strong>Duvidas</strong> para consultas rapidas e <strong>Suporte</strong> quando houver necessidade de atendimento.
</Callout>
