## Problema

Ao tentar utilizar o certificado A3 no Windows 11, o sistema apresenta o erro **2148073481**.

<Callout type="warn">
  Este erro está relacionado à compatibilidade do driver do certificado A3 com o Windows 11. Verifique se o driver está atualizado para a versão mais recente antes de seguir os passos abaixo.
</Callout>

## Solução

1. Acesse o **Gerenciador de Dispositivos** do Windows.
2. Localize o leitor de cartão ou token A3 na lista de dispositivos.
3. Clique com o botão direito e selecione **Atualizar driver**.
4. Reinicie o computador após a atualização.
5. Tente utilizar o certificado novamente no Syspro ERP.

Se o problema persistir, entre em contato com o suporte técnico informando o modelo do token/cartão e a versão do driver instalado.
