## Trilhas de Treinamento Syspro ERP

Bem-vindo à nossa central de treinamentos. Diferente dos manuais de referência, esta seção foi projetada para ensinar **processos de negócio completos**, do início ao fim. Cada card abaixo representa uma "trilha de aprendizado", um guia passo a passo que levará você através de diversas telas e módulos para completar uma tarefa real do dia a dia.

Se você é um novo usuário ou está aprendendo uma nova função, comece por aqui.
