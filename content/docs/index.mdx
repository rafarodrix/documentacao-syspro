## Introdução

**Syspro ERP** é um sistema de planejamento de recursos empresariais projetado para gerenciar diversas operações de uma empresa de forma eficiente e integrada. Ele é composto por vários módulos que atendem a funções específicas, como cadastro de dados, gestão de estoque, compras, finanças, movimentos, questões fiscais, contabilidade, produção, vendas e relatórios.

### Terminologia

**Módulo:** Cada módulo no Syspro ERP é uma unidade funcional independente que pode ser configurada e utilizada conforme as necessidades da sua empresa.

Ter um conhecimento básico da interface do usuário e dos processos de negócios será útil para personalizações adicionais.

## Acesso Inicial

Para começar a usar o Syspro ERP, faça login com suas credenciais fornecidas pelo administrador do sistema.

<Callout title="Primeira vez no sistema?">
  Certifique-se de que suas credenciais estejam corretas e entre em contato com o suporte se precisar de assistência.
</Callout>

### Comece a Usar!

Acesse o módulo **Cadastro** para adicionar seus primeiros dados.

Por exemplo, para cadastrar um cliente:

1. Navegue até o módulo **Cadastro**.
2. Selecione "Clientes" e clique em "Adicionar Novo".
3. Preencha os campos como nome, endereço e detalhes de contato.
4. Salve o registro.

Agora você está pronto para explorar os outros módulos!

## Explorando o Sistema

No Syspro ERP, os módulos são organizados de forma intuitiva:

- **Painel Principal:** O ponto de entrada onde você pode acessar todos os módulos.
- **Barra Lateral:** Oferece navegação rápida para os módulos mais usados.
- **Módulos Individuais:** Cada um contém ferramentas específicas para suas funções.

| Seção             | Descrição                                      |
|-------------------|------------------------------------------------|
| `Painel Principal`| Visão geral e acesso aos módulos disponíveis. |
| `Módulos`         | Ferramentas específicas para cada área da empresa. |

## Personalizando os Módulos

Os módulos do Syspro ERP podem ser configurados para atender às suas necessidades. Por exemplo:

- No módulo **Cadastro**, adicione campos personalizados para informações específicas.
- Ajuste preferências como formatos de data e moeda em cada módulo.

Saiba mais em [Personalização de Módulos](/docs/personalizacao).

## Personalizando a Navegação

A barra lateral pode ser ajustada para incluir atalhos aos módulos mais utilizados. Use a função de busca para encontrar rapidamente dados ou funcionalidades específicas.

## Inserindo Dados

Cada módulo possui interfaces dedicadas para entrada de dados:

- **Cadastro:** Formulários para adicionar clientes, fornecedores ou produtos.
- **Venda:** Crie pedidos selecionando produtos e aplicando descontos.

Veja mais detalhes em [Como Inserir Dados](/docs/dados).

## Personalizando a Interface

Ajuste a aparência do Syspro ERP escolhendo temas de cores ou reorganizando os painéis para otimizar seu fluxo de trabalho.

Saiba mais em [Personalização da Interface](/docs/interface).
