## Introdução

Este guia rápido é o seu ponto de partida para o Syspro ERP. O objetivo aqui é proporcionar uma introdução clara e direta sobre como acessar o sistema e se familiarizar com sua interface principal. Dominar estes primeiros passos é fundamental para que você possa explorar os demais módulos com confiança, garantindo uma rápida adaptação e o máximo aproveitamento da plataforma desde o primeiro dia.

O Syspro ERP foi projetado para integrar e otimizar todas as áreas da sua empresa, desde o cadastro de clientes até a geração de relatórios estratégicos.

## Conhecendo a Interface Principal

Após o login, você terá acesso à tela principal do Syspro ERP. Entender seus componentes é o primeiro passo para navegar com eficiência. Abaixo, detalhamos as três áreas mais importantes que você encontrará.

<Cards>
  <Card title="Painel Principal (Dashboard)">
    Sua central de comandos. Aqui você encontra atalhos, indicadores de performance e um resumo das atividades importantes, oferecendo uma visão geral do seu negócio.
  </Card>
  <Card title="Barra de Módulos">
    Localizada geralmente na lateral ou no topo, esta barra é o seu mapa para todas as funcionalidades do sistema. A partir dela, você acessa áreas como `Cadastro`, `Vendas`, `Financeiro`, etc.
  </Card>
  <Card title="Área de Trabalho">
    É o espaço central da tela onde as funcionalidades do módulo selecionado são exibidas. É aqui que você preencherá formulários, visualizará listas e executará as tarefas do dia a dia.
  </Card>
</Cards>

## Passo a Passo: Primeiro Acesso e Navegação

Siga estas etapas para realizar seu primeiro acesso e começar a explorar o Syspro ERP.

<Steps>
  <Step>
    ### 1. Acesse o Sistema
    Utilize o link de acesso fornecido pelo administrador do sistema para abrir a tela de login do Syspro ERP no seu navegador.
  </Step>
  <Step>
    ### 2. Faça o Login
    Insira seu nome de usuário e senha nos campos indicados e clique em "Entrar". Mantenha suas credenciais em um local seguro.
  </Step>
  <Step>
    ### 3. Explore o Painel Principal
    Ao entrar, dedique um momento para observar o Painel Principal. Identifique os atalhos e os indicadores disponíveis. Este será seu ponto de partida diário.
  </Step>
  <Step>
    ### 4. Navegue até um Módulo
    Use a Barra de Módulos para acessar sua primeira tarefa. Por exemplo, navegue até `Cadastro > Clientes` para visualizar a lista de clientes existentes ou adicionar um novo.
  </Step>
</Steps>

<Callout title="Dica de Produtividade">
Você pode personalizar a barra de módulos ou o painel principal para exibir as funcionalidades que mais utiliza, agilizando seu fluxo de trabalho.
</Callout>

## Perguntas Frequentes

<Accordions>
  <Accordion title="Esqueci minha senha. O que faço?">
    Na tela de login, procure pela opção "Esqueci minha senha" ou entre em contato diretamente com o administrador do sistema na sua empresa para solicitar a redefinição.
  </Accordion>
  <Accordion title="Posso personalizar a aparência do sistema?">
    Sim, o Syspro ERP permite ajustes na interface, como a escolha de temas de cores e a reorganização de elementos no painel, para adequar o sistema ao seu gosto e necessidade. Consulte a seção de configurações de usuário.
  </Accordion>
  <Accordion title="Como encontro uma funcionalidade específica?">
    Utilize a barra de busca global, geralmente localizada no topo da tela. Ela permite pesquisar por menus, funcionalidades ou até mesmo dados específicos (como o nome de um cliente) em todo o sistema.
  </Accordion>
  <Accordion title="Onde começo a inserir os dados da minha empresa?">
    O ponto de partida ideal é o módulo de **Cadastro**. Comece cadastrando informações essenciais como `Clientes`, `Fornecedores` e `Produtos`. Ter esses dados bem estruturados é a base para utilizar os outros módulos.
  </Accordion>
</Accordions>
