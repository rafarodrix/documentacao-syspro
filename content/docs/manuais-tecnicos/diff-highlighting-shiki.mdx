## Objetivo

Mostrar mudancas de codigo com clareza em guias de migracao e changelogs.

## Como usar

Use marcadores de comentario dentro do bloco de codigo:

- `// [!code ++]` para linha adicionada
- `// [!code --]` para linha removida
- `// [!code highlight]` para linha de destaque neutro

## Exemplo

```ts
const config = {
  apiUrl: process.env.API_URL, // [!code --]
  apiUrl: process.env.NEXT_PUBLIC_API_URL, // [!code ++]
  timeout: 8000, // [!code highlight]
};
```

## Boas praticas

1. Use diff highlight somente quando houver comparacao de antes/depois.
2. Evite marcar muitas linhas no mesmo bloco.
3. Combine com texto curto explicando o motivo da mudanca.
