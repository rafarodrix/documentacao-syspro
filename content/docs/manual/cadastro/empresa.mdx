O cadastro de empresas é essencial para o funcionamento do sistema, pois todas as operações dependem das informações registradas. Este manual visa apresentar as configurações básicas e necessárias para o funcionamento do sistema.

> **Recomendação:**
> Preencha todas as informações, incluindo aquelas que não são obrigatórias, pois algumas funcionalidades do sistema dependem de dados adicionais para operar corretamente.

A seguir, apresentamos um guia detalhado para o cadastro de empresas, garantindo que todas as informações relevantes sejam incluídas, proporcionando um desempenho otimizado do sistema.

### **Caminho**

Acesse o caminho abaixo para adicionar um novo cadastro de empresa:

`Cadastros > Empresa`

### **Cadastro da Empresa**

Nesta seção, é possível configurar os dados cadastrais da empresa no **Syspro ERP**.

* **Código**: Código único da empresa.
* **Empresa Participante**: Cadastro da empresa. `Deve estar previamente cadastrado no módulo de participantes.`
* **Fantasia**, **CNPJ**, **Inscrição Estadual**, **Cód. Suframa**, **Inscrição Municipal**: carregados automaticamente do participante.
* **Regime (CRT)**: Selecione entre **Simples Nacional**, **Simples c/ Excesso Sublimite**, **Regime Normal**.
* **Empresa Matriz**: Informe se é matriz (mesmo cadastro) ou filial (informe a matriz).
* **Participante Responsável** e **Contador**: Devem estar cadastrados no módulo de participantes.
* **Logotipo**: Imagem da empresa para exibição em documentos.

### **Aba Outros**

#### Guia Precificação

Esta guia permite configurar como será feita a precificação de produtos no sistema. A escolha correta desses parâmetros afeta diretamente os preços de venda e as análises de margem de lucro.

- **Tipo Precificação**: Escolha a metodologia utilizada para formação de preço:
  - **Margem Líquida**
  - **Margem Bruta**

- **Custo Precificação**: Define qual custo será utilizado como base para a precificação:
  - **Reposição**
  - **Aquisição**

- **Data Precificação**: Selecione se a data será atualizada automaticamente ao calcular preços:
  - **Sim**
  - **Não**

- **Código Custo**: Informe um código de custo específico, se necessário.

- **Venda Sugestão**: Define se o sistema deve sugerir preços de venda:
  - **Sim**
  - **Não**

##### Sugestão Cadastro Produto

- **Lucro Bruto Projetado (%)**: Percentual de margem bruta estimada sobre o custo.
- **Lucro Líquido (%)**: Percentual líquido final projetado após impostos e despesas.

##### Verificação de Lote

- **Verifica Lote XML**: Se ativado, o sistema analisará os XMLs de entrada em busca de informações de lote. Caso detectado, marcará automaticamente o produto como controlado por lote.
  - **Sim**
  - **Não**
