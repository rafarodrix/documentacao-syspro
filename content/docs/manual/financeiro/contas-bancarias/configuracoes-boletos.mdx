## <Building className="inline-block mr-2" /> Visão Geral

A configuração de contas bancárias no **Syspro ERP** é o que chamamos de "ponte de confiança". Sem esses dados validados, o fluxo de caixa trava. Este guia foca na **homologação**, o processo onde garantimos que o banco "entende" o arquivo que o sistema gera.

<BankingExport />

<Callout type="warn">
**Atenção:** Nunca inicie a emissão em produção sem a confirmação de leitura (liquidação) de um arquivo de teste pelo suporte do banco.
</Callout>

## <FileText className="inline-block mr-2 mb-1" /> Dicionário de Dados Técnicos

Utilize estes termos técnicos para alinhar a configuração com o seu gerente de conta.

| Campo | O que é? | Por que é importante? |
| :--- | :--- | :--- |
| **Código do Cedente** | Sua identidade no banco. | Identifica quem deve receber o dinheiro. |
| **Carteira** | O "tipo" de cobrança. | Define se o boleto terá protesto automático ou desconto. |
| **CNAB 400/240** | O layout do arquivo. | Formato do arquivo de remessa enviado ao banco. |
| **CNAB 444** | Layout para Antecipação. | **Obrigatório** para enviar a Chave da NF-e para descontos. |
| **Nosso Número** | O RG do boleto. | Garante que um boleto nunca seja igual ao outro. |

## <Send className="inline-block mr-2 mb-1" /> Fluxo de Homologação

<Steps>
<Step>
### Coleta de Dados
Solicite ao banco os dados de homologação de cobrança registrada (CNAB 240/400).
</Step>
<Step>
### Cadastro no ERP
Insira os códigos no módulo `Financeiro > Configurações > Contas Bancárias`. Atenção redobrada aos dígitos verificadores.
</Step>
<Step>
### Teste de Remessa
Gere um arquivo `.REM` com títulos de teste e envie ao portal do banco para validação de layout.
</Step>
<Step>
### Verificação de Leitura
Imprima um boleto e valide se o código de barras aponta para o beneficiário correto (sua empresa).
</Step>
</Steps>

## <HelpCircle className="inline-block mr-2 mb-1" /> Perguntas Frequentes

<Accordions>
<Accordion title="Por que usar o CNAB 444?">
Necessário para empresas que fazem antecipação de títulos, pois permite trafegar a chave da NF-e exigida pela legislação.
</Accordion>
<Accordion title="Como evitar duplicidade de boletos?">
Configure o campo "Próxima Faixa" no Syspro com o número subsequente ao último boleto gerado no seu sistema antigo.
</Accordion>
</Accordions>
